# 2sec

# 1日を「2秒」で残す。一番シンプルなライフログカメラ。

> 「旅先で撮った長い動画，結局見返していますか？」 撮りっぱなしで眠っている動画たち。録画停止ボタンを押す手間。 2secは，そんな悩みから生まれた「2秒間だけ」を切り取るカメラアプリです。
> 

### ダウンロード

ダウンロードは，[こちら](https://apps.apple.com/jp/app/google-%E3%82%A2%E3%83%97%E3%83%AA/id284815942)（iOS）。

### 2sec の特徴

1. **迷わず撮れる「上下分割」UI**
    
    アプリを開けば，そこが撮影スタジオ。複雑な設定は一切ありません。
    
    **画面上部をタップ： 外カメラで，目の前の景色を。**
    
    **画面下部をタップ： 内カメラで，あなたの表情を。**
    
    タップすると2秒のカウントダウン後に撮影開始。「録画停止」を押す必要はありません。2秒経てば自動で終わります。
    
2. **翌日0:10のお楽しみ**
    
    今日撮った2秒間のクリップたちは，翌日の0:10に自動で1本の動画としてまとまります。 「動画の準備ができました！」の通知が届いたら，昨日のハイライトを楽しむ時間です。
    
3. **「見返せる」長さに収まります**
    
    1日に30回撮影しても，できあがる動画はたったの「1分」。 忙しい日々の中でも，移動中や休憩時間にサクッと見返せる。それが2secの体験です。
    

### 基本的な使い方

**① 撮影する** メイン画面の上半分（外カメ）か下半分（内カメ）をタップして撮影します。 撮影後はプレビューが表示され，以下の操作が可能です。

- 撮り直す： 気に入らなければすぐにリテイク。
- もう1本撮る： 続けてどんどん撮影。
- 削除： プレビュー右上の「×」ボタンで削除も簡単。

**② エクスポート** 通常は翌日の通知を待ちますが，**「今すぐまとめたい」**時も大丈夫。 ホーム画面右上のアイコンから，その時点での「今日のまとめ動画」をすぐに作成・保存できます。

**③ 過去の動画** 動画はアプリ内に**7日間**保存されます。 「あの日の動画を保存し忘れた」と思っても，6日前までなら日付を選択してエクスポート可能です。
